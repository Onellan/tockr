# UI Impact Plan

## Additions

- Account dropdown links to Profile/Security.
- Profile page with profile, password, and TOTP sections.
- Calendar nav item with read-only weekly agenda.
- Workspace switcher in topbar when multiple workspaces are available.
- Project Members route from project row overflow.
- Rates form includes task and effective date fields plus user cost entry.

## Deferred

- Collapsible sidebar: current menu density does not justify the added state and discoverability cost.
- Drag/drop calendar editing.
- Full account preferences/theming.
