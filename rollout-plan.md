# Rollout Plan

## Phase 1

- Favorites / quick timer starts.
- Tasks under projects.
- Project estimates and fixed-fee budget basics.
- Project dashboard basics.
- Report filters and saved reports.

## Phase 2

- Richer group/project membership editors.
- Approvals and locked time periods.
- Billable rate improvements and labor cost basics.
- Profitability reports.
- Project templates.

## Phase 3

- Historical rates.
- Scheduled reports.
- Workload/utilization views.
- More webhook events and integration endpoints.
- Calendar-assisted time entry.

## Rollback

All Phase 1 schema changes are additive. Rollback remains restore-from-backup for production SQLite databases.
