# UI Optimization Plan

## Focus areas delivered

1. Foundations
   - Engineer-facing navigation labels
   - Cleaner workflow grouping
   - Consistent operational copy

2. Time capture core
   - Dashboard quick-start and reuse flows
   - Timesheet prefill via query parameters
   - Recent/favorite shortcuts
   - Weekly review emphasis
   - Readable calendar context

3. Delivery clarity
   - Clients framing for billing entities
   - Work Types framing for activities
   - Enriched project and project dashboard visibility
   - Clearer project/group membership copy

4. Billing and analysis
   - Billable filter in reports
   - Better saved-report persistence for the new filter
   - Stronger invoice/rate/admin language

## Design rules used

- Fast repeated time entry beats decorative UI.
- Tables, badges, and contextual shortcuts beat heavy charting.
- Keep the app bright, compact, and Pi-friendly.
- Additive changes only; preserve existing behavior where possible.
