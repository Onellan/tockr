# Component Inventory

## Shell

- Sidebar navigation grouped into Work, Manage, Analyze, and Admin.
- Topbar with current workspace context and logout.
- Responsive collapse into stacked navigation on smaller screens.

## Page Patterns

- `page-head`: eyebrow, title, description.
- `panel`: framed form or operational section.
- `panel-head`: title and guidance inside panels.
- `metric-card`: dashboard summary values.
- `table-card`: dense responsive table wrapper.
- `empty-state`: consistent no-records state.

## Controls

- Primary, danger, ghost, and table action buttons.
- Form grids for create/edit flows.
- Toolbar forms for compact single-line actions.
- Badges for status and boolean states.
- Tabs for report grouping.
